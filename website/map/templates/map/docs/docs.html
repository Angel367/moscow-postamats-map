Краткое руководство пользователя

Для пользования картой, перейдите в раздел карта на сайте решения. Затем в панели фильтров выберите:

  1. Требуемый радиус для расчета
  2. Район(ы) и/или округ(а)
  3. Типы объектов для размещения постаматов
  4. Модель расчета востребованности

После этого необходимо запустить расчет, в следствии чего можно отфильтровать результаты расчета и выгрузить данные о точках в нужном формате:

  • Таблица MS Excel
  • Документ PDF
  • Статическое изображение карты (с параметрами фильтра или без)
